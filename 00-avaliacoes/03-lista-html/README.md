# Lista de Exercícios de HTML

## Tags Semânticas

1. Crie um documento HTML5 básico que contenha as tags semânticas `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>` e `<footer>`.
2. Utilize a tag `<article>` para criar um artigo de blog com um título, parágrafo de introdução, corpo do texto e um parágrafo de conclusão.
3. Implemente uma página web que use a tag `<nav>` para criar um menu de navegação com links para seções diferentes da página.

---

## Parágrafos

4. Crie um parágrafo com texto fictício usando a tag `<p>`.
5. Adicione dois parágrafos ao seu documento HTML e utilize a tag `<br>` para inserir uma quebra de linha em um dos parágrafos.
6. Utilize a tag `<blockquote>` para adicionar uma citação famosa no seu documento.

---

## Imagens

7. Insira uma imagem no seu documento HTML usando a tag `<img>` e atribua um texto alternativo apropriado.
8. Utilize as tags `<figure>` e `<figcaption>` para adicionar uma imagem com uma legenda.
9. Implemente uma página com três imagens alinhadas horizontalmente. Use a propriedade `style` para ajustar o tamanho das imagens.

---

## Listas

10. Crie uma lista ordenada de três itens usando a tag `<ol>`.
11. Crie uma lista não ordenada de três itens usando a tag `<ul>`.

---

## Tabelas

12. Crie uma tabela simples com três colunas e três linhas usando as tags `<table>`, `<tr>`, `<th>` e `<td>`.
13. Adicione um cabeçalho à tabela criada na questão anterior usando a tag `<thead>`.
14. Adicione um rodapé à tabela criada na questão anterior usando a tag `<tfoot>`.

---

## Formulários

15. Crie um formulário com campos de texto para o nome e email do usuário.
16. Adicione ao formulário criado na questão anterior um botão de envio usando a tag `<button>`.
17. Crie um formulário com um campo de seleção (dropdown) para escolher o gênero (Masculino, Feminino, Outro).
18. Adicione ao formulário um grupo de botões de rádio para selecionar a faixa etária do usuário.
19. Implemente um formulário com um campo de texto, um campo de senha e uma área de texto para comentários.
