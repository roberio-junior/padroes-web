# Questões

## Questão 1: Estrutura Básica de HTML e CSS

Crie um documento HTML básico (`index.html`) com a estrutura a seguir:

- Um cabeçalho (`<header>`) com um título (`<h1>`) que diz **"Avaliação Prática de HTML e CSS"**.
- Uma navegação (`<nav>`) com três links: **Home**, **Sobre**, **Contato**.
- Um conteúdo principal (`<main>`) com uma seção (`<section>`) contendo um artigo (`<article>`) com:
  - Um título (`<h2>`) **"Introdução"**
  - Um parágrafo (`<p>`) com texto fictício.
- Um rodapé (`<footer>`) com o texto **"Todos os direitos reservados"**.

---

## Questão 2: Estilização Básica com CSS

Crie um arquivo CSS (`styles.css`) e adicione o link para este arquivo no documento HTML.

Adicione as seguintes regras de estilo:

- Altere a cor de fundo do corpo da página para um cinza claro (`#f0f0f0`).
- Centralize o texto do cabeçalho (`<header>`).
- Defina a cor do texto do título do artigo como azul.
- Adicione uma margem de `20px` para o conteúdo principal (`<main>`).

---

## Questão 3: Classes e IDs

Adicione classes e IDs aos elementos HTML e defina estilos específicos no CSS:

- Adicione uma classe `menu` à navegação (`<nav>`) e defina no CSS:
  - Cor de fundo azul
  - `padding` de `10px`

- Adicione um ID `principal` ao conteúdo principal (`<main>`) e defina no CSS:
  - Borda de `2px` sólida preta

---

## Questão 4: Listas e Imagens

No conteúdo principal (`<main>`), adicione:

- Uma lista não ordenada (`<ul>`) com três itens:
  - HTML
  - CSS
  - JavaScript

- Uma imagem ilustrativa abaixo da lista.

Estilização:

- Defina a cor do texto dos itens da lista (`<ul>`) como verde.
- Adicione à imagem:
  - Largura de `300px`
  - Borda de `1px` sólida preta

---

## Questão 5: Formulários

No rodapé (`<footer>`), adicione um formulário simples para contato:

O formulário deve conter:

- Um campo de texto para **nome**
- Um campo de **email**
- Um botão de envio

Estilização:

- Os campos do formulário devem ter:
  - `padding` de `5px`
  - `margin` de `10px` entre eles

- O botão de envio deve ter:
  - Cor de fundo verde
  - Cor do texto branca
